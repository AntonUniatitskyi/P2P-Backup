\## Установка



Скачайте установщик из \[последнего релиза](../../releases/latest) —

`P2PBackupSetup.exe` — и запустите. Ничего дополнительно ставить не нужно.



\## Разработка



Если хочешь собрать проект из исходников или контрибьютить:



\\`\\`\\`bash

git clone ...

invoke setup

invoke gui

\\`\\`\\`



Требуется Python 3.12+, Go 1.21+ — но только для разработки,

конечным пользователям это не нужно.
